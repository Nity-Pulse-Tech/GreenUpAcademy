Trouvez des réponses aux questions courantes sur les admissions et les études à Green Up Academy

Demandez un visa auprès du consulat de France dans votre pays. Consultez le site web du consulat pour connaître les procédures spécifiques et les délais de traitement, qui peuvent prendre plusieurs mois. Les documents requis varient selon le pays et le type de visa. Note : Votre passeport doit être valide pour toute la durée de votre séjour en France, sinon votre demande sera rejetée.

Visitez Campus France pour les procédures de visa étudiant, qui varient selon le pays :

  • Procédure pour 33 pays avec inscription en ligne (CEF)
  • Procédure pour les pays européens
  • Procédure pour les autres pays

Lors de la soumission de votre candidature, deux possibilités :

Si Green Up Academy est répertoriée dans votre pays : Sélectionnez notre institution, suivez les étapes et fournissez le pack visa délivré par Green Up Academy lors de l'inscription.

Si Green Up Academy n'est pas répertoriée : Entrez "Green Up Academy" dans le champ de texte et joignez le pack visa fourni par l'école.

Campus France vérifiera votre inscription auprès de nos services.

Les admissions sont ouvertes pour les sessions d'octobre 2025 et février 2026. La date limite de candidature est un mois avant la date de début. Dates clés :

  • 15 octobre 2025
  • 10 février 2026
